# 🧮 React Calculator

A simple and responsive calculator built using **React**, **Vite**, **JavaScript**, and **Tailwind CSS**.  
This project demonstrates component-based UI design, state management, and modern frontend tooling.

---

## 🚀 Tech Stack

- ⚛️ React
- ⚡ Vite
- 🟨 JavaScript (ES6+)
- 🎨 Tailwind CSS

---

## ✨ Features

- Basic arithmetic operations (+, −, ×, ÷)
- Clear (C) and delete functionality
- Responsive design for mobile and desktop
- Clean and modern UI using Tailwind CSS
- Fast development with Vite
